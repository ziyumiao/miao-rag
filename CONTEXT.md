# oprag - 售前键帽客服

基于 LangGraph + LlamaIndex 的键帽售前智能客服系统，支持多轮对话、轴体识别、订单查询、人工转接。

## Language

**键帽 (Keycap)**:
安装在键盘轴体上的塑料外壳，用户的可购买商品。本店只售卖键帽，搭配赠送拔键器。
_Avoid_: 按钮、帽、键盘帽

**轴体 (Switch)**:
键盘按键的机械结构，键帽安装的底座。键帽兼容性取决于轴体的机械结构。
_Avoid_: 开关、按键轴

**MX 结构 (MX Structure)**:
Cherry 发明的一种十字柱轴心结构，是行业标准。注意：标准 MX 轴与矮轴（Low Profile）虽然都是十字柱，但键帽接口物理尺寸不同，键帽不互通。
_Avoid_: 十字轴、十字芯

**标准 MX 轴 (Standard MX Switch)**:
传统高度的 MX 结构轴体，键帽接口深度标准。本店当前全部键帽均适配此类轴体。
_Avoid_: 普通轴、正常轴

**矮轴 (Low Profile Switch)**:
高度较薄的 MX 结构轴体，十字柱与标准 MX 相同但卡槽更浅，需专用键帽。后续商品线。
_Avoid_: 薄轴、短轴

**键盘类型 (Keyboard Type)**:
机械键盘（使用独立机械轴体）、磁轴键盘（使用霍尔效应轴）、薄膜键盘（使用橡胶碗）、静电容键盘（使用电容感应）。机械和磁轴大部分为十字轴心。
_Avoid_: 键盘种类

**材质 (Material)**:
键帽的塑料材质，常见 PBT（耐磨不油）和 ABS（手感光滑易油）。
_Avoid_: 原料、塑料

**高度 (Profile)**:
键帽的外形弧度与高度。常见：OEM、原厂 (Cherry)、DSA、XDA、SA、MDA。
_Avoid_: 形状、弧度

**客户 (Customer)**:
通过旺旺/千牛咨询键帽的买家，系统可通过 `buyer_nick` 查询其历史订单。
_Avoid_: 用户、买家、买家昵称

**订单 (Order)**:
客户的淘宝购买记录，通过 `taobao.trades.sold.get` API 获取，包含 SKU、购买时间等。
_Avoid_: 交易、购买记录

**追问 (Follow-up)**:
Agent 发现信息不足时向客户提出的问题，单轮对话内不超过 3 次，超过则触发人工转接。
_Avoid_: 多轮追踪、回问

**人工转接 (Escalation)**:
Agent 无法解决问题时将对话转交给真人客服，携带完整对话摘要。
_Avoid_: 转人工、升级、转交

**聊天记录 (Chat Record)**:
历史客户与客服的旺旺对话，是知识库的核心数据源之一。通过规则 + LLM 清洗提取 QA 对和事实。
_Avoid_: 对话历史、客服记录

**清洗流水线 (Cleaning Pipeline)**:
聊天记录 → 规则提取（70%）+ LLM 提取（30%）→ QA 对 + 事实三元组 → 入库的批处理流程。
_Avoid_: 数据预处理、ETL
